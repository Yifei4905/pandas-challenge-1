# E-Commerce Data Analysis

This project analyzes a fictional e-commerce dataset using Python's Pandas library to identify top customers, popular product categories, and calculate financial metrics.

## Features

- **Data Exploration**: Import data, view column names, and gather basic statistics.
- **Top Categories and Clients**: Identify top item categories, subcategories, and clients.
- **Data Transformation**: Calculate subtotals, shipping prices, total prices, costs, and profits.
- **Data Verification**: Confirm calculations with provided order receipts.
- **Summary and Analysis**: Summarize financial metrics for top clients and format data for presentation.

## How to Use

1. **Clone the Repository**:
    ```bash
    git clone <repository-url>
    ```
2. **Install Dependencies**:
    ```bash
    pip install pandas
    ```
3. **Run the Analysis Script**:
    ```bash
    python analysis.py
    ```
